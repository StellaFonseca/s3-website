Site hospedado em Bucket S3

Exemplo básico de demonstração de habilidades em nível iniciante com AWS. Projeto de hospedagem de um site estático em um bucket S3, utilizando o AWS Management Console.

Requisitos:

  1. Arquivo HTML/CSS (estático)
  2. Conta ativa na AWS
  3. Bucket S3 com configurações de políticas
  4. AWS Management Console ou AWS CLI configurado
